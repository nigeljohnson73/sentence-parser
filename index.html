About this application

		
No electrons were created or destroyed in the making of this webiste, however some were significantly mamed.

		
This site is the product of a project created to generate a simple sentence parser. The GitHub project can be
		found here. The idea was that I 
		wanted a really simple and lightweight sentence parser that didn't need sending off to the NLP labs at Berkley. 
		I needed to extract a few bits from the sentence so a response could be formulated. The output would then be used 
		in my Nige Knows website.

		
The Google fonts API supplied the "Open Sans" fonts 
		that are used throughout the site.

		
Ace provided the code renderificator used to modify the text 
		area for the javascript code in the parsing structure section.

		
The overall style was extracted from the JQuery 
		Mobile demos website and modified a bit from there.
